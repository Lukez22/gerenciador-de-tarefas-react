# Gerenciador de Tarefas

Aplicação web desenvolvida utilizando React para gerenciar tarefas do dia a dia.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Concluir tarefas
- Remover tarefas
- Tema claro e escuro
- Layout responsivo
- Persistência de dados com localStorage

## Tecnologias utilizadas

- React
- JavaScript
- HTML
- CSS

## Como rodar o projeto

Clone o repositório:

git clone https://github.com/Lukez22/gerenciador-de-tarefas-react.git

Instale as dependências:

npm install

Execute o projeto:

npm run dev
